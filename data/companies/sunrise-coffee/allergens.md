# Allergen Information

> **Note:** If a customer expresses concern about a specific allergen or has a severe allergy, the agent must escalate to staff immediately. Never guess — always confirm with the team.

## Allergen Table

| Item               | Dairy | Gluten | Nuts | Soy |
|--------------------|-------|--------|------|-----|
| Oat Milk Latte     | No    | No*    | No   | No  |
| Classic Cappuccino | Yes   | No     | No   | No  |
| Chai Latte         | Yes** | No     | No   | No  |
| Matcha Latte       | No**  | No     | No   | No  |
| Cold Brew          | No    | No     | No   | No  |
| Iced Vanilla Latte | Yes   | No     | No   | No  |
| Avocado Toast      | No    | Yes*** | No   | No  |
| Blueberry Muffin   | Yes   | Yes    | No   | No  |

**Notes:**
- `*` Oat milk may contain trace gluten. Customers with celiac disease should be informed and escalate.
- `**` Chai Latte and Matcha Latte use dairy by default; request oat/almond/soy milk for dairy-free.
- `***` Avocado Toast uses sourdough (contains gluten). Gluten-free bread available on request.

All items are prepared in a kitchen that also handles dairy, gluten, and tree nuts. Cross-contamination is possible. For severe allergies, escalate to staff.
