# HOG Runtime Seed Data

These CSV files are the runtime copies of the HOG Costco and Starbucks demo catalogs.

- Costco and Starbucks are ingested as separate company brains.
- The top-level `hog-data/` folder mirrors these files for clear repository provenance.
- Seed taxonomy fields are demo enrichments and may be inferred.
- Pricing, stock, allergen, and policy details are demo data unless explicitly sourced elsewhere.
